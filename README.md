# 🧠 Simulador de Razonamiento Lógico y Matemático

¡Bienvenido al desafío mental definitivo para la preparación universitaria! Este es un juego interactivo web inspirado en el formato clásico de *"¿Quién Quiere Ser Millonario?"*, diseñado específicamente para entrenar la agilidad cognitiva, el razonamiento matemático y la lógica formal.

## 🚀 Características del Proyecto
* **Banco de 100 Preguntas Reales:** Dividido en categorías clave como Secuencias Numéricas, Alfabéticas, Alfanuméricas, Lógica Proposicional, Silogismos y Planteamiento Matemático.
* **Sistema de 4 Comodines Activos:**
    * **50/50:** Elimina dos opciones incorrectas.
    * **Eliminar 1:** Quita una respuesta errónea al azar.
    * **📞 Llamada al Experto:** Una simulación interactiva que te da un consejo lógico con un 95% de acierto.
    * **👥 Votación del Público:** Despliega una gráfica de barras en tiempo real con la tendencia de la sala.
* **Retroalimentación Inmediata:** Cada pregunta muestra una explicación detallada del porqué de la respuesta correcta al ser contestada.

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica del simulador.
* **CSS3:** Estilos avanzados, gradientes radiales y animaciones de juego.
* **JavaScript (ES6+):** Lógica del juego, motor de aleatoriedad para las preguntas y control de estados de los comodines.
