# 🧠 Simulador de Razonamiento Lógico-Matemático Universitario

Este repositorio contiene el código completo y corregido de un simulador web interactivo basado en el formato clásico de *"¿Quién Quiere Ser Millonario?"*, adaptado con un banco de **100 preguntas reales y variadas** de aptitud académica para exámenes de ingreso a la universidad.

## 🚀 Contenido del Paquete de Archivos:
- `index.html`: Estructura e interfaz semántica del juego.
- `index.css`: Hoja de estilos con paleta oscura y diseño responsivo premium.
- `index.js`: Motor dinámico del simulador que gestiona la aleatoriedad, los comodines interactivos y almacena los **100 ejercicios reales únicos corregidos**.
- `README.md`: Este documento explicativo.
